# Create React App using Typescript
## Firstly
```
npx create-react-app . --template typescript
```
## Libraries used
- ### `Material UI`
``` 
npm i @material-ui/core @material-ui/icons 
```
- ### `react query`
``` 
npm i react-query 
```
- ### `styled-components`
``` 
npm i styled-components @types/styled-components
```
